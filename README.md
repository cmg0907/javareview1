### IF 문 

## 기본예시

```
if (조건문) {
    <수행할 문장1>
    <수행할 문장>
    ...
}else {
    <수행할 문장A>
    <수행할 문장B>
    ...
}
```



## 비교연산자

```
비교연산자	설명
x < y	x가 y보다 작다
x > y	x가 y보다 크다
x == y	x와 y가 같다
x != y	x와 y가 같지 않다
x >= y	x가 y보다 크거나 같다
x <= y	x가 y보다 작거나 같다
```
